# Brainshare

## Features

- [x] upload csv file
- [x] identify columns
- [x] suggest widgets
- [x] custom types
- [ ] public types
- [ ] export widgets as ipynb
- [ ] export widgets as excel

## Development Setup

After cloning the repository, navigate to the `frontend` directory and run `npm install` to install frontend dependencies and set up the pre-commit hooks:

```bash
cd frontend
npm install
```
